Installation Guide

To install beginner.js on a website, you can download the source here, or install it directly using script tags that look like:

<script src="https://raw.githubusercontent.com/alaskamani15/beginner.js/master/src/src.js"></script>
<script>
Library.greet();
</script>

Installed Functions:

stop("Now!");

The stop() function creates an alert box with the text given.

Library.greet();

Useful for debugging to make sure the library is active. It should print out: Hello from the library.

kill();

A random function that will kill the browser. Just for fun. The JavaScript equivalent is: (note: the kill(); function will ask for a confirmation. For no confirmation, use the evil(); function) var questki = confirm("This will kill the browser! Click OK to continue."); if (questki == true){ var i = 0; while (i < Infinity){ console.log("Please wait!"); //i++ } } else { console.log("Cancelled Kill"); } }
